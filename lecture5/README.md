

# 第5回：勾配法と誤差逆伝播法

## 到達目標
* 基本的な勾配降下法のアルゴリズムを理解する
* ロジスティック回帰における勾配法を実装できる
* フィードフォワードニューラルネットワーク及び誤差逆伝播法の仕組みを理解する
* 指導を受けながら、誤差逆伝播法の実装を体験する

## キーワード
* 勾配降下法
* クロスエントロピー誤差
* フィードフォワードニューラルネットワーク
* 誤差逆伝播法
* 確率的勾配降下法 (SGD)
* ミニバッチ確率的勾配降下法 (MSGD)
