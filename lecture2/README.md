# 第2回：機械学習入門

## 到達目標
* 機械学習の目的及び種類とその評価方法を理解する。
* 第1回に引き続き、numpyで頻出する関数の使い方を実践的に理解する。
* k-NN法を理解し、それをnumpyを用いて実装できる。

## キーワード
* 教師あり学習
* 教師なし学習
* 強化学習
* 訓練データ、テストデータ、バリデーションデータ
* 交差検証 (CV)
* 適合率、再現率、F値、正解率
* 過学習
* 汎化誤差
* np.sum, np.mean, np.var
* np.random
* Advanced Indexing
* np.sort, np.argsort, np.argpartition
* k-Nearest Neighbor (k-NN) 法
* コサイン類似度
