# 第3回：線形回帰

## 到達目標
* Pythonの利便性を上げるクラスなどの構文が書けるようになる
* 二乗誤差最小化に基づく線形回帰アルゴリズムを数式で記述し、その実装ができる
* 実データを用いて線形回帰を行い、その結果をMatplotlibを用いて可視化できる

## キーワード
* 最小二乗回帰
* map, reduce, filter
* Advanced indexing
* Matplotlib
